# c2l Skill Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a Codex `c2l` skill that runs the chess2lichess CLI, enriches the result with optional `lichess` data, and gives Codex a clean structure for Korean game explanations.

**Architecture:** Keep the skill thin and deterministic. Put execution, discovery, JSON parsing, and enrichment into a single Python helper script, then keep `SKILL.md` focused on trigger conditions, workflow, and explanation rules.

**Tech Stack:** Codex skill format, Python 3 standard library, local `c2l` CLI, optional `lichess` HTTP endpoints, `unittest`
